# Práctica 02 — Terreno y Navegación

## Descripción

Este proyecto implementa un controlador de personaje en tercera dimensión utilizando Unity, incluyendo:

* Movimiento propio del Agente
* Movimiento con click izquierdo a posición deseada

---

## Características

### Movimiento

* Click izquierdo → posición deseada

---

## Requisitos

Unity 6 o superior

Input System Package instalado

---

## Instrucciones de uso

1. Clonar repositorio

2. Abrir proyecto en Unity Hub

3. Abrir escena:

Scenes/Practica01_3D.unity

4. Presionar Play
